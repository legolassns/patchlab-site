Request a Quote

We don't need precise details before understanding your project. Send us what you have: artwork, an approximate quantity and the intended use. We'll start from there.

What helps us prepare your proposal

  • Logo or artwork — vector (AI, EPS, PDF) or high resolution (300 dpi)
  • Approximate quantity — even a range (e.g. "between 100 and 200 pieces")
  • Desired dimensions — in millimetres or with a physical reference
  • Intended use — type of garment, material, environment of use
  • Approximate date of use — to assess timing
  • Application type — sew-on, iron-on, hook-and-loop, or to be defined

Don't have all these details? No problem — send us what you have. We'll work out the rest together.

After you submit

What happens after you submit your request

No automatic reply, no generic quote. Five real steps.

01
Receipt

We read your request

Your request is read by a person, not filtered by an automated system. If we're missing details we need to assess it, we'll write back with specific questions.

02
Analysis

We assess the artwork

If you've already sent us an artwork file — even by email, as the form doesn't yet handle attachments — we analyse it against the requested dimensions: text legibility, colours, details. If you haven't sent it yet, we'll ask for it in our reply. If the file isn't suited to the chosen technique, we'll tell you before going ahead.

03
Proposal

We send you a proposal

You'll receive a proposal with the recommended technique, price and timing. If the technique you requested isn't the most suitable for your project, we'll explain why — along with the alternatives.

04
Sample

We produce the sample

After the proposal is approved, we produce the physical sample. We never start batch production before the sample has been approved.

05
Production

We produce and archive

Production takes place after the sample is approved. Once finished, we archive the machine file and specifications to handle future reorders without starting from scratch.
