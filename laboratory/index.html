Production

Inside the workshop

PatchLab was born from Euroricami's production experience: machinery, digitising, threads, sampling and technical archiving work together to turn a design into a real patch.

Technical expertise

Machines aren't enough

An embroidery machine executes instructions. Good instructions come from digitising — the skilled translation of a design into sequences of stitches, tensions and thread directions.

The same machine, with two different files, produces two different patches. One precise and solid. The other full of flaws that don't show up until batch production.

The technical expertise of whoever prepares the machine file is what separates a workshop from a machine operator.

Structure

The patch department

Not an add-on service. A dedicated department within Euroricami.

Sampling

Before starting batch production, we make the physical sample from a validated file. The customer approves it — or requests changes — on the real piece, not a digital simulation.

In-house digitising

The machine file is prepared in-house for the techniques we handle. It's the most technical step in the whole process: a faulty file produces a faulty result at any price.

Quality control

Every patch is checked before delivery. Colours, dimensions, border and backing are compared against the approved sample. We don't deliver non-conforming pieces.

Technical archive

The machine file, approved sample and technical specifications are archived after every order. A reorder starts again from this data — without redoing the digitising or the sample, if the specifications haven't changed.

Production process

From file to patch

Six technical steps, each with an impact on the final result.

01
Input

Artwork

The customer's artwork file is analysed relative to the patch's final size. Text legibility, number of colours, design complexity: all of this determines whether the chosen technique is feasible — and whether adjustments are needed.

02
Digitising

Machine file

The design is translated into machine instructions: stitch sequence, directions, tensions, colours, stops. This is the stage that requires the most technical expertise — and the one with the greatest impact on the final result.

03
Materials

Threads and base

Threads are selected based on the design's colours. The base — twill, felt or otherwise — is chosen according to the required aesthetic and the intended use.

04
Sample

First production

The sample is produced from the final machine file. It's the physical piece the customer sees, touches and approves before batch production begins.

05
Border and backing

Final touches

After production, every patch receives its border — merrowed, laser-cut or otherwise — and the backing agreed with the customer: sew-on, iron-on, hook-and-loop (hook side) or combinations.

06
Inspection

Quality control

Every batch produced is checked before delivery. The comparison is made against the approved sample — not abstract parameters. Non-conforming pieces are removed before packaging.

Quality

What we check before delivery

Inspection isn't a bureaucratic step. It's what separates a good batch from one that needs redoing.

Checkpoints

  • Colours — matching the approved sample
  • Dimensions — conforming to the project specifications
  • Border — intact, even, with no gaps or irregularities
  • Backing — correctly applied and uniform
  • Loose threads — no visible untrimmed threads
  • General cleanliness — no moulding defects, scorching or production residue

Pieces that fail inspection aren't delivered. The batch is topped up or restarted before shipping.

Archive

Every project leaves a trace

After every completed order, we archive the final machine file, a copy of the approved sample, the technical specifications and relevant production notes.

This data doesn't expire. A customer who returns after two years to reorder the same patch finds the files exactly where we left them.

✓

What this means for reorders

If the specifications don't change, the reorder starts from the file already validated. There's no need to pay for digitising again, remake the sample, or start from scratch. The process is faster and less costly than the first order.

If the specifications change — different dimensions, updated artwork, a new backing — we work out together what needs redoing and what can be reused from the archive.
